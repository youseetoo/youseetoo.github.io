Fields tagged Edit are the ones you usually change. Gray fields are prefilled but still editable.
Export uses an embedded known-good IAA-Plus template and only replaces values inside it. Import fills the form from a selected XML.
This page prepares the XML for IAA-Plus. It does not generate the ABD itself. MRN and ABD are created by ATLAS only after the declaration has been accepted and released.

Header

Procedure

Choose the export declaration type for the customs workflow. For plain two-stage standard export use 00000100. Use 00000200 only if you also apply for presentation outside the customs office.
Format for customs offices: country code + 6-digit office number, e.g. DE003004.

Dates

Auto-set: submission = now, presentation = next day 07:00, loading end = next day 13:00. Adjust if needed.
