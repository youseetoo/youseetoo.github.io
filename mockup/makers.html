Hardware: 3D-printable cubes

All cube designs are available as STL/STEP files on GitHub. Print them on any FDM printer. Injection-molded cubes available in the shop for higher precision.

⚡

Electronics: UC2-ESP32

Custom PCB with ESP32 microcontroller, motor drivers, LED control, and CAN bus. Open schematics and Gerber files. Or use off-the-shelf components.

Firmware & schematics →
💻

Software: ImSwitch

Modular microscopy control in Python. GUI, REST API, Jupyter integration. Supports USB3 and CSI cameras, motorized stages, and custom hardware.

ImSwitch on GitHub →

What people build with openUC2

🧬 Custom fluorescence microscope

A PhD student in Jena built a 3-color fluorescence microscope for under €2,000 using 3D-printed cubes and surplus optics.

🐟 Light-sheet for zebrafish

A developmental biology lab assembled a dual-illumination light-sheet system from UC2 modules for live zebrafish embryo imaging.

🏥 Point-of-care prototype

A startup prototyped a portable diagnostic microscope using UC2 cubes before moving to injection-molded production parts.
