Introducing FRAME v2: What's New

2026-03-01Dr. Benedict Diederich

We are excited to announce FRAME v2, the latest iteration of our professional automated microscope platform.

What changed

  • Faster autofocus — the new z-stage controller reduces focus-lock time from 800 ms to under 200 ms.
  • LED matrix illumination — a programmable 8×8 LED array enables brightfield, darkfield, DPC, and FPM from a single source.
  • Improved thermal management — passive heatsinks and an optional fan bracket keep long time-lapse experiments stable.

Software improvements

ImSwitch 2.1 ships alongside FRAME v2 with:

  • Rebuilt experiment designer with drag-and-drop tile creation
  • Native OME-ZARR export for large datasets
  • One-click system health dashboard

Availability

FRAME v2 kits are available now on shop.openuc2.com. Existing FRAME v1 owners can order the upgrade module separately.

Stay tuned for detailed benchmarks and comparison data in the coming weeks.
